TSP Solver MLOps Platform: Architectural Design

This application contrasts conventional graph heuristics against a sequence-to-sequence neural model to solve the 2D Traveling Salesman Problem.

1. Neural Network Architecture

The neural solver uses a Pointer Network model in PyTorch. Key characteristics:

  • Input Representation: City nodes are represented as 2D coordinate vectors $(x, y) \in [0, 1]^2$. They are projected into a 128-dimensional hidden space using a Linear node embedding.
  • Transformer Encoder: A 3-layer Transformer Encoder processes the sequence of embedded coordinates, computing multi-head self-attention to capture spatial context and global structure.
  • Pointer Decoder: Generates the route path. At each step, a query vector (concatenating the current node, start node, and global average embedding) is projected against the encoder's outputs. Softmax scores pointing to unvisited nodes dictate the next city to visit.
  • Loss Function: Categorical Negative Log-Likelihood (NLL) loss. The network is trained with Teacher Forcing using paths solved by Google OR-Tools.

2. Conventional Solvers

  • Google OR-Tools: Uses Guided Local Search (GLS) meta-heuristics to find high-quality solutions, acting as the ground-truth trainer.
  • Christofides Algorithm: Constructs a minimum spanning tree, finds minimum weight perfect matchings on odd vertices, constructs a Eulerian tour, and shortcuts it. Guarantees a path length within 1.5x of the true optimum.
  • Nearest Neighbor: A greedy heuristic. Starts at city 0, iteratively jumps to the closest unvisited city, and returns to 0.
  • Held-Karp Exact Solver: A dynamic programming algorithm to find the absolute mathematical shortest path. Runs in $O(2^N N^2)$ complexity, limited to $N \le 10$ to prevent browser crashes.

3. Closed-Loop Telemetry & MLOps

Production runs logged to MongoDB Atlas/SQLite collect coordinate structures. The retraining loop automatically takes logged production data, retrains the Pointer Network, and reloads the serving weights to iteratively minimize the accuracy gap.
